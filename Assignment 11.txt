2) Write a query that produces the names and ratings of all customers who have above-average orders.
- SELECT cname, rating
    -> FROM customer
    -> WHERE cnum IN (
    ->     SELECT cnum
    ->     FROM orders
    ->     GROUP BY cnum
    ->     HAVING AVG(amt) > (
    ->         SELECT AVG(amt)
    ->         FROM orders
    ->     )
    -> );
